# Книга рецептов (Recipe Book) - сервис для хранения и поиска кулинарных рецептов.

## Основная идея:
Пользователь может создавать, хранить и искать рецепты (ну и само собой удалять). Ключевая фишка — поиск рецептов по имеющимся у пользователя ингредиентам ("Что можно приготовить из того, что есть в холодильнике?").

## Алгоритм, который мы хотим сделать:

Получаем список ID ингредиентов пользователя (например, [1, 3, 5, 7]).

Для каждого рецепта получаем список ID требуемых ингредиентов (из RecipeIngredient).

Проверяем, что все требуемые ID входят в множество ID пользователя.

## Пример
### Ингредиенты:

ID 1: Яйцо (DAIRY)  
ID 2: Молоко (DAIRY)  
ID 3: Мука (OTHER)  
ID 4: Картофель (VEGETABLE)  
ID 5: Мясо (MEAT)

### Рецепты

Рецепт "Омлет":  
Title: "Омлет классический"  
RecipeIngredient:  
    ingredientId: 1 (Яйцо), quantity: 2, unit: PIECE  
    ingredientId: 2 (Молоко), quantity: 50, unit: MILLILITER

Рецепт "Картофельное пюре":  
Title: "Пюре с мясом"  
RecipeIngredient:  
    ingredientId: 4 (Картофель), quantity: 500, unit: GRAM  
    ingredientId: 5 (Мясо), quantity: 200, unit: GRAM

### Как это работает:

Если пользователь пришлет ingredients=1,2 (яйца и молоко), API вернет только "Омлет".

Если пользователь пришлет ingredients=1,2,3,4,5 (все), API вернет оба рецепта.

Если пользователь пришлет ingredients=4,5 (картошка и мясо), API вернет только "Пюре с мясом".

## Если слишком быстро управимся
Категории рецептов: Завтрак, Обед, Ужин, Десерт.  
Избранное: Пользователь может добавлять рецепты в "Избранное".  
Диеты: Поддержка фильтрации по диетам (веганское, безглютеновое и т.д.).
